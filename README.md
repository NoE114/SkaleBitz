# SkaleBitz

**A Web app** 

[SkaleBitZ](https://skalebitz.vercel.app/) is a trust-first fintech platform that helps MSMEs raise capital and investors deploy funds with real-time risk and performance visibility.

## Features

- Lightweight and optimized for performance.
- 99.9% JavaScript, ensuring compatibility and maintainability.
- Extensible and easy to integrate into any JavaScript-based project or ecosystem.

## Getting Started

### Prerequisites

Make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/en/download) (v24 or later recommended)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/NoE114/SkaleBitz.git
   ```

2. Open 1st terminal & setup Backend:

   ```bash
   cd SkaleBitz
   cd Back
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Open 2nd terminal & setup Backend::

   ```bash
   cd SkaleBitz
   cd Front
   ```

5. Install dependencies:

   ```bash
   npm install
   ```

### Usage

To start the development server use this cmd (both terminal):

```bash
npm run dev
```
